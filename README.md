# Madhi Foundation Task

An Android application built with Kotlin as part of the Madhi Foundation assignment, demonstrating clean architecture and modern Android development practices.

## Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM
- **UI:** Material Design

## Getting Started

1. Clone the repository
```
git clone https://github.com/bhartendukodes/MadhiFoundationTask.git
```
2. Open in Android Studio
3. Build and run
